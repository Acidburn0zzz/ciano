<h2> </h2>

Cyan is a simple multimedia file converter. Built from the ground up for the elementary operating system.


### Features

- Video 
	- [ ] MP4
	- [ ] 3GP
	- [ ] MPG
	- [ ] AVI
	- [ ] WMV
	- [ ] FLV
	- [ ] SWF

- Audio
	- [ ] MP3
	- [ ] WMA
	- [ ] AMR
	- [ ] OGG
	- [ ] AAC
	- [ ] WAV

- Images
	- [ ] JPG
	- [ ] BMP
	- [ ] PNG
	- [ ] TIF
	- [ ] ICO
	- [ ] GIF
	- [ ] TGA
